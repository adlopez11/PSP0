Tarea 1 PSP0

DESCARGA

Para descargar el proyecto desde el repositorio de GitHub, seguir los siguientes pasos:

- Abrir en una consola o shell del sistema
- Ubicarse en la raiz de la carpeta de proyectos
- Ejecutar el siguiente comando:

git clone https://github.com/adlopez11/psp0.git proyectopsp0

Se creará una carpeta llamada "proyectopsp0" con todo el contenido del repositorio, se puede cambiar el nombre de la carpeta.


EJECUCIÓN

Luego de compilar el proyecto, ubicados en la raiz del proyecto, la aplicación se puede ejecutar con el comando:

java -cp target/psp0-1.0-SNAPSHOT.jar edu.uniandes.ecos.App

La aplicación preguntará por un archivo plano que se debe colocar en la ruta /src/test/resources/
